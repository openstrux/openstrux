# Openstrux

Openstrux is a standard for certifiable AI-native software.
Structure first. Code second. Trust built in.

## Repositories

| Repo | Purpose |
|---|---|
| [openstrux](https://github.com/openstrux/openstrux) | Manifesto, governance, benchmarks |
| [openstrux-spec](https://github.com/openstrux/openstrux-spec) | Normative language specification |
| [openstrux-core](https://github.com/openstrux/openstrux-core) | Parser, validator, AST, IR |

## Principles

- AI-native
- Token- and cost-efficient
- Certified by design
- Human-translatable on demand
- Built for performance
- Structure first. Code second.
- Trust built in. Not bolted on.
