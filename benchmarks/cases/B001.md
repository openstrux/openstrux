# B001 — Typed Record

**id:** B001
**category:** token-efficiency

## Intent Prompt

Define a `GrantApplication` record type with 6 fields: `id` (string), `title` (string), `amount` (number, 0..1000000), `status` (string), `submittedAt` (date), `applicantId` (string). The record models a grant application entity.

## Openstrux Source

```
@type GrantApplication {
  id:          string,
  title:       string,
  amount:      number [0..1000000],
  status:      string,
  submittedAt: date,
  applicantId: string
}
```
