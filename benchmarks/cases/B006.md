# B006 — Filter Expression Shorthand

**id:** B006
**category:** token-efficiency
**constructs:** expression shorthand (`filter`, `arg.predicate`)

## Intent Prompt

Write a panel named `list-pending-applications` that reads all grant applications where status is `submitted` AND amount is between 10000 and 500000 from a Postgres database named `@db`, and responds with the results. Use filter expression shorthand.

## Openstrux Source

```
@panel list-pending-applications {
  @access { purpose: "grant_review", operation: "read" }

  src  = read-data { source: @db, mode: scan, predicate: sql: SELECT * FROM grant_applications }
  flt  = filter    { predicate: status == "submitted" AND amount BETWEEN 10000 AND 500000 }
  _    = respond   { status: 200 }
}
```

## TypeScript Baseline

```typescript
// Next.js App Router route handler
import { NextRequest, NextResponse } from "next/server";
import { prisma } from "@/lib/prisma";

export async function GET(_req: NextRequest): Promise<NextResponse> {
  const applications = await prisma.grantApplication.findMany({
    where: {
      status: "submitted",
      amount: {
        gte: 10000,
        lte: 500000,
      },
    },
  });

  return NextResponse.json(applications, { status: 200 });
}
```
