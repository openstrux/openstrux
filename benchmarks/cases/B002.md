# B002 — Enum Type

**id:** B002
**category:** token-efficiency
**constructs:** `@type` enum

## Intent Prompt

Define a `GrantStatus` enum with 5 variants: `draft`, `submitted`, `under_review`, `approved`, `rejected`. This models the lifecycle states of a grant application.

## Openstrux Source

```
@type GrantStatus = enum { draft, submitted, under_review, approved, rejected }
```

```typescript
// TypeScript enum
export enum GrantStatus {
  Draft        = "draft",
  Submitted    = "submitted",
  UnderReview  = "under_review",
  Approved     = "approved",
  Rejected     = "rejected",
}
```

